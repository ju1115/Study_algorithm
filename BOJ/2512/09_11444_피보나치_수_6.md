- 수학적 해결이 필요함. 행렬연산
- fn+1   [ 1 1 ] fn
- fn   = [ 1 0 ] fn-1
- 이 행렬 곱을 반복하면
- fn+1   [ 1 1 ]^n f1
- fn   = [ 1 0 ]   f0
- 해당 행렬식을 n번곱한 값이 결과가 된다.
