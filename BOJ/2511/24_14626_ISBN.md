# 📝 ISBN (BOJ 14626)

### 💡 접근 방식 (Idea)

- Char array로 받아 13자리중 \*인 부분을 저장하고 모두 더함 나머지로 해당 자리의원래 값을 찾음

### ⚠️ 막혔던 부분 / 중요 포인트

- \*이 0이면 10-나머지가 10이됨 이값또한 나머지 연산을 해야한다.
- gpt 는 이런 역산방식보단 브루트포스 0~9까지 모두더해 답을 찾는 방식을 추천함

### 🗝️ 해결 코드 (Key Point)

```java
        ans = ans % 10;
        ans = (10 - ans) % 10;
        if (num % 2 == 1) {
            ans = check(ans);
        }

    public static int check(int n) {
        for (int i = 0; i <= 9; i++) {
            if ((i * 3) % 10 == n)
                return i;
        }
        return -1;
    }
```
