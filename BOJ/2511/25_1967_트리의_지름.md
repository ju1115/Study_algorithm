# 📝 트리의\_지름 (BOJ 1967)

### 💡 접근 방식 (Idea)

- n-1 개의 행이 들어옴 부모 자식 가중치
- 리프 노드 두개를 당겼을 때 최장길이, 두 노드의 첫 번째 공통 부모 노드까지의 가중치 합이 될것.
- 1번노드 가중치 0 자식노드는 자신 부모에서 자신까지의 가중치 합을 가지고 있는다.
- 리프를 nC2로 비교

### ⚠️ 막혔던 부분 / 중요 포인트

- 모르면 풀 수 없다.
- 트리의 지름을 구하기 위해선 하나의 임의 노드를 고르고 가장 먼 노드를 찾는다.
- 가장 먼노드의 가장 먼 노드까지의 거리가 지름이 된다.
- 노드번호 순서대로 주어져서 루트부터 리프 순으로 주어지는 줄 알았으나 노드번호와 순서는 관계가 없다.

### 🗝️ 해결 코드 (Key Point)

```java
// 핵심 로직만 복사해서 기록해두기
    static void dfs(int now, int curV) {
        visited[now] = true;

        if (maxV < curV) {
            maxV = curV;
            endP = now;
        }

        for (int i = 0; i < adj[now].size(); i++) {
            Node next = adj[now].get(i);
            if (visited[next.to])
                continue;
            dfs(next.to, curV + next.v);
        }
    }
```
