# 📝 팩토리얼3 (BOJ 27434)

### 💡 접근 방식 (Idea)

- 배열로 큰 수 연산을 저장한다. 10^5 이 100000 이라 가정 배열크기 500000으로 1부터 5까지 곱한다.

### ⚠️ 막혔던 부분 / 중요 포인트

- 각 자리수에 그냥 곱하고 올림처리를 매번 해줘야함
- BigInteger라는 라이브러리를 자바가 제공

### 🗝️ 해결 코드 (Key Point)

```java
    // 1. BigInteger 초기화 (값: 1)
    BigInteger ans = BigInteger.ONE;

    // 2. 1부터 N까지 곱하기
    for (int i = 1; i <= N; i++) {
        // ans = ans * i; 와 같음
        ans = ans.multiply(BigInteger.valueOf(i));
    }
```

```java
    static void resize(int[] ans) {
        for (int i = 0; i <= idx; i++) {
            if (ans[i] >= 10) {
                ans[i + 1] += ans[i] / 10; // 몫을 다음 자리로 넘김
                ans[i] %= 10; // 나머지만 남김

                // 자릿수가 늘어났다면 idx 증가
                if (i == idx && ans[i + 1] > 0)
                    idx++;
            }
        }
    }

    static void times(int[] now, int v) {
        for (int i = 0; i <= idx; i++) {
            now[i] *= v; // 현재 자릿수에 곱함
        }
        resize(now);
    }
```
